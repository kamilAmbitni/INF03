Obsługa MySQL w PHP – mysqli oraz PDO

W tej lekcji poznasz dwa nowoczesne interfejsy API do obsługi baz danych w PHP: mysqli oraz PDO.

  • mysqli – obsługuje MySQL, dostępny w wersji proceduralnej i obiektowej.
  • PDO – obiektowy interfejs obsługujący wiele silników SQL (np. MySQL, SQLite).
