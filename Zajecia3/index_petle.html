Pętle w JavaScript — proste demo

Przykłady: for, while, do..while, break, continue, forEach().

Pętla for — znamy liczbę iteracji


      

      

        
        

      

      

        
        

      

      

        Składnia: for (start; warunek; krok) { /* ciało pętli */ }

        • Inkrementacja: i++ (zwiększ o 1) • Dekrementacja: i-- (zmniejsz o 1) • Krok: np. i += 2.
      

    


    


    
    

      
Pętla while — dopóki warunek jest spełniony

      
      

      

        Pętla while wykonuje się tak długo, jak warunek jest prawdziwy. Tu pytamy o liczbę aż trafisz (lub anulujesz).
      

    


    


    
    

      
Pętla do..while — co najmniej jedno wykonanie

      
      

      

        do { /* ciało */ } while (warunek); — warunek sprawdzany na końcu, więc ciało wykona się minimum raz.

continue — pomiń bieżącą iterację

      
      

      
Jeśli element jest pustym tekstem "" — pomijamy go i jedziemy dalej.

    


    


    
    

      
forEach() — iteracja po elementach tablicy

      
      

      

        tablica.forEach((element, indeks) => { ... }) — wygodny sposób na przejście po wszystkich elementach.
