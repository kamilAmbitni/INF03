Zmiana stylów elementów w JavaScript (style)

W JS każdy uchwycony element ma właściwość style. Przez nią zmieniamy CSS „w locie”. Uwaga: nazwy z myślnikiem w CSS (np. background-color) w JS zapisujemy w camelCase: backgroundColor, fontSize, textAlign, itd.

1) Zmień kolor i rozmiar tekstu (prosty przykład)

To jest przykładowy akapit. Kliknij przyciski poniżej, aby zmienić styl.

CSS  →  JS (camelCase)
background-color → backgroundColor
font-size        → fontSize
text-align       → textAlign
margin-top       → marginTop
border-radius    → borderRadius

2) Warunkowa zmiana stylu pola (zielony <=> czerwony)

Reguła: jeśli liczba > 100 → tło zielone, w przeciwnym razie czerwone.

3) „Ciemny motyw” (zmień tło i kolor tekstu całej strony)

Zmieniamy style na document.body.style: backgroundColor, color.
